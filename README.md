# 🎫 MyTicketApp – מערכת ניהול טיקטים

## 🧾 תיאור המערכת
MyTicketApp היא מערכת לניהול פניות (טיקטים) בין לקוחות, סוכנים ומנהל מערכת.

המערכת מאפשרת:
- פתיחת טיקטים
- צפייה ומעקב אחר סטטוס
- שיוך טיקטים לסוכנים
- הוספת תגובות
- ניהול משתמשים וסטטוסים

המערכת פועלת עם הרשאות שונות לפי סוג המשתמש.

הפרויקט נכתב ב־**React + TypeScript**, משתמש ב־**React Router** לניווט  
וב־**Context API** לניהול התחברות והרשאות, עם תקשורת ל־Backend באמצעות REST API.

---

## 👥 תפקידי משתמשים

### 🛠 Admin (מנהל)
- צפייה בכל הטיקטים במערכת
- ניהול טיקטים (שינוי סטטוס, שיוך לסוכן)
- ניהול סטטוסים
- יצירת משתמשים חדשים (Admin / Agent / Customer)
- לוח בקרה ייעודי למנהל

### 🎧 Agent (סוכן)
- צפייה בטיקטים שהוקצו אליו בלבד
- עדכון סטטוס טיקטים
- הוספת תגובות
- לוח בקרה מותאם לסוכן

### 🙋 Customer (לקוח)
- יצירת טיקט חדש
- צפייה בטיקטים שיצר בלבד
- צפייה בפרטי טיקט והתגובות
- הוספת תגובות
- לוח בקרה מותאם ללקוח

---

## 🔐 התחברות ואבטחה
- התחברות מתבצעת באמצעות JWT
- הטוקן נשמר ב־`localStorage`
- בעת רענון הדף מתבצעת קריאה ל־`/auth/me` לשחזור המשתמש
- ניווט בין עמודים מוגן לפי תפקיד המשתמש
